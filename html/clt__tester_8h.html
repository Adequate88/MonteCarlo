Detailed Description

Defines the CltTester class, which tests the Central Limit Theorem (CLT) using sample data.

This file contains the definition of the CltTester class. The CltTester class is used to demonstrate and validate the Central Limit Theorem by generating multiple samples and calculating the mean and standard deviation of those sample means.
