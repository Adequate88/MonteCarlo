Detailed Description

Defines the classes and functions used for parsing mathematical expressions into an Abstract Syntax Tree (AST).

This file contains definitions for tokens, AST nodes, helper functions for creating AST nodes, and a Parser class for converting mathematical expressions into an AST representation that can be used for evaluation or other analyses.
