\doxysubsection{Detailed Description}
Abstract base class for sampling and distribution generation. 

The {\ttfamily \doxylink{class_abstract_sampler}{Abstract\+Sampler}} class provides a virtual interface for generating distributions and sampling values. Derived classes should override the virtual methods to implement specific sampling logic.
