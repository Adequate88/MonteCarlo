\doxysubsection{Detailed Description}
Class that tests the Central Limit Theorem using an abstract sampler. 

This class takes in an abstract sampler and generates multiple sets of samples, calculates their means, and then checks if the mean and standard deviation of these sample means conform to the expected properties defined by the Central Limit Theorem.

This method takes multiple sets of N samples, calculates their means, and stores these sample means in the {\ttfamily sample\+\_\+means\+\_\+} vector. It then divides the sample means into bins and calculates the frequency of each bin to populate the {\ttfamily distribution\+\_\+array}. The plot range (minimum and maximum) is also determined from the sample means.
