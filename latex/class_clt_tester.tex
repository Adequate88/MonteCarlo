\doxysubsection{Detailed Description}
Class that tests the Central Limit Theorem using an abstract sampler. 

This class takes in an abstract sampler and generates multiple sets of samples, calculates their means, and then checks if the mean and standard deviation of these sample means conform to the expected properties defined by the Central Limit Theorem.
