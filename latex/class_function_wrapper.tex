\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Input\+Type, typename Output\+Type$>$\newline
class Function\+Wrapper$<$ Input\+Type, Output\+Type $>$}
Wrapper class that adapts a standard C++ function to an \doxylink{class_abstract_function}{Abstract\+Function} object. 

This class allows you to wrap a regular C++ function (such as a lambda or function pointer) and use it as an \doxylink{class_abstract_function}{Abstract\+Function} object. This is particularly useful for integrating custom functions into a broader framework that expects functions derived from \doxylink{class_abstract_function}{Abstract\+Function}.
