\doxysubsection{Detailed Description}
Generates samples from a normal distribution using the Box-\/\+Muller method. 

Extends the \doxylink{class_normal_sampler}{Normal\+Sampler} class to provide an implementation of the Box-\/\+Muller transform for generating normal-\/distributed random numbers.
