\hypertarget{md__r_e_a_d_m_e_autotoc_md423}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{md__r_e_a_d_m_e_autotoc_md423}
This project provides a modular framework for sampling, statistical analysis, and testing mathematical properties such as the Central Limit Theorem. It includes\+:


\begin{DoxyItemize}
\item Sampling distributions (e.\+g., uniform, normal, and Box-\/\+Muller transforms).
\item A mathematical expression parser
\item Statistical computations such as expectation, variance, and moments.
\item Distribution Plotting and saving in build file
\item A Central Limit Theorem tester.
\end{DoxyItemize}

\item Run the executable.

\begin{DoxyEnumerate}

\item {\bfseries{{\ttfamily sampler\+\_\+method}}}\+: Choose the random number generator method ({\ttfamily inverse\+\_\+transform\+\_\+sampling}, {\ttfamily box\+\_\+muller\+\_\+transform}, {\ttfamily uniform\+\_\+distribution}).
\item {\bfseries{{\ttfamily number\+\_\+of\+\_\+samples}}}\+: Specify the number of random samples to generate.
\item {\bfseries{{\ttfamily random\+\_\+seed}}}\+: Set an integer seed for the random number generator (optional).
\item {\bfseries{{\ttfamily histogram\+\_\+bins}}}\+: Set the number of bins for the histogram (0 to disable).
\item {\bfseries{{\ttfamily compute\+\_\+expectation\+\_\+value}}}\+: Set to {\ttfamily yes} to compute the expectation value of {\ttfamily f(x)}.
\item {\bfseries{{\ttfamily statistical\+\_\+moments}}}\+: Provide a list of statistical moments to compute (e.\+g., {\ttfamily 1,2,3} for mean, variance, and skewness).
\item {\bfseries{{\ttfamily verify\+\_\+central\+\_\+limit\+\_\+theorem}}}\+: Set to {\ttfamily yes} to verify the Central Limit Theorem.
\item {\bfseries{{\ttfamily output\+\_\+file}}}\+: Specify the output file name for saving results.

\end{DoxyItemize}
\item {\bfseries{Parentheses}}\+:
\begin{DoxyItemize}

\begin{DoxyItemize}

\end{DoxyItemize}
\item {\bfseries{Combination of Functions and Operations}}\+:
\begin{DoxyItemize}

\hypertarget{md__r_e_a_d_m_e_autotoc_md434}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md434}
\hypertarget{md__r_e_a_d_m_e_autotoc_md435}{}\doxysubsection{\texorpdfstring{Sampling Module}{Sampling Module}}\label{md__r_e_a_d_m_e_autotoc_md435}
Classes in this module are based on an \doxylink{class_abstract_sampler}{Abstract\+Sampler} class. This abstract class provides the basic framework common to all other sampling classes, such as\+:


\begin{DoxyItemize}
\item {\ttfamily sample()} method to obtain a random value.
\item {\ttfamily generate\+Distribution(int bins, int n\+\_\+samples)} method to generate the probability density function of the sampler. This is then stored into a {\ttfamily std\+::vector\texorpdfstring{$<$}{<}double\texorpdfstring{$>$}{>}}.
\item {\ttfamily print\+Distribution()} and {\ttfamily plot\+Distribution()} to visualize the distribution generated.
\end{DoxyItemize}

The currently available sampling methods found here are\+:
\begin{DoxyItemize}
\item {\bfseries{Uniform Distribution}} Sampling ({\ttfamily \doxylink{class_uniform_sampler}{Uniform\+Sampler}})\+: Algorithm obtained from "{}\+Numerical Recipes"{}
\item {\bfseries{Normal Distribution}} (with parent class {\ttfamily \doxylink{class_normal_sampler}{Normal\+Sampler}})\+:
\begin{DoxyItemize}
\item Box-\/\+Muller Transform ({\ttfamily \doxylink{class_normal_box_muller_sampler}{Normal\+Box\+Muller\+Sampler}})\+: Generating pairs of independent, normally distributed random variables from uniformly distributed random variables. Involves\+:
\begin{DoxyEnumerate}
\item Generate two independent random variables, {\ttfamily u1} and {\ttfamily u2}, uniformly distributed between 0 and 1.
\item Apply the Box-\/\+Muller formulas to convert these variables into two independent random variables with a standard normal distribution (mean 0, variance 1)\+:
\begin{DoxyItemize}
\item z0 = sqrt(-\/2 \texorpdfstring{$\ast$}{*} ln(u1)) \texorpdfstring{$\ast$}{*} cos(2 \texorpdfstring{$\ast$}{*} pi \texorpdfstring{$\ast$}{*} u2)
\item z1 = sqrt(-\/2 \texorpdfstring{$\ast$}{*} ln(u1)) \texorpdfstring{$\ast$}{*} sin(2 \texorpdfstring{$\ast$}{*} pi \texorpdfstring{$\ast$}{*} u2)
\end{DoxyItemize}
\item By scaling and shifting (by mean/mu and standard deviation/sigma respectively), these can be adjusted to any desired mean and variance.
\end{DoxyEnumerate}
\item {\bfseries{Inverse Transform}} ({\ttfamily \doxylink{class_normal_inverse_sampler}{Normal\+Inverse\+Sampler}})\+: Generating normally distributed random variables using the inverse cumulative distribution function (CDF). Involves\+:
\begin{DoxyEnumerate}
\item Generate a uniform random variable, {\ttfamily u}, uniformly distributed between 0 and 1.
\item Apply the inverse CDF (also known as the quantile function) to convert {\ttfamily u} into a value from a standard normal distribution (mean 0, variance 1).
\item By scaling and shifting (by mean/mu and standard deviation/sigma respectively), the result can be adjusted to any desired mean and variance.
\end{DoxyEnumerate}
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md436}{}\doxysubsection{\texorpdfstring{Statistics Module}{Statistics Module}}\label{md__r_e_a_d_m_e_autotoc_md436}
By taking in an \doxylink{class_abstract_sampler}{Abstract\+Sampler} object, this module will compute the desired statistics whilst sampling from that sampler. The supported statistical methods take in a user-\/defined \doxylink{class_abstract_function}{Abstract\+Function} object to compute\+:


\begin{DoxyItemize}
\item {\bfseries{Expectation}}\+: Sums all obtained samples (after applying function {\ttfamily f(x)}) and then divides by number of samples
\item {\bfseries{Variance}}\+: Computes expectation of the {\ttfamily f(x)}. Then, for each value in data, the squared difference from the expectation is summed. The total is divided by number of samples.
\item {\bfseries{Nth Moment}}\+: Function {\ttfamily f(x)} is applied to each sample, and the difference from the mean is raised to the Nth power. The moments are computed by summing these values and dividing by number of samples.
\item {\bfseries{Nth Central Moment}}\+: Unlike the regular moment, this does not subtract the mean before raising to the Nth power. The sum of the values raised to the power is computed and divided by number of samples.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md437}{}\doxysubsection{\texorpdfstring{Central Limit Theorem Tester (Included as an Abstract\+Sampler Child)}{Central Limit Theorem Tester (Included as an Abstract\+Sampler Child)}}\label{md__r_e_a_d_m_e_autotoc_md437}
The CLT Tester module validates the Central Limit Theorem (CLT) for a given sampler and statistical function by analyzing the distribution of sample means.


\begin{DoxyItemize}
\item {\bfseries{Sampling and Means}}\+: Computes means from repeated samples using an {\ttfamily \doxylink{class_abstract_sampler}{Abstract\+Sampler}} and a user-\/defined {\ttfamily \doxylink{class_abstract_function}{Abstract\+Function}}.
\item {\bfseries{Histogram Generation}}\+: {\ttfamily generate\+Distribution(int bins, int n\+\_\+samples)} creates a frequency distribution of sample means, normalized to a probability density function.
\item {\bfseries{CLT Validation}}\+: One can plot the obtained distribution histogram to verify that a normal distribution has been obtained.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md438}{}\doxysubsection{\texorpdfstring{Functions Module}{Functions Module}}\label{md__r_e_a_d_m_e_autotoc_md438}
This template based module is comprised of two parts\+: the \doxylink{class_abstract_function}{Abstract\+Function} class, meant to hold and represent functions to use in our statistics calculations; and a \doxylink{class_parser}{Parser} class, capable of breaking down a string mathematical expression into its primitive functions and then constructing an \doxylink{class_abstract_function}{Abstract\+Function} object to represent that expression.


\begin{DoxyEnumerate}
\item {\bfseries{\doxylink{class_abstract_function}{Abstract\+Function} Class}}\+:
\begin{DoxyItemize}
\item Serves as the base class for different types of mathematical functions.
\item Defines the {\ttfamily eval} method as a pure virtual function, requiring derived classes to implement their own evaluation logic.
\item Supports function composition, addition, multiplication, and exponentiation through overloaded operators ({\ttfamily \texorpdfstring{$\vert$}{|}}, {\ttfamily +}, {\ttfamily \texorpdfstring{$\ast$}{*}}, {\ttfamily \texorpdfstring{$^\wedge$}{\string^}}).
\item {\bfseries{Children Function Classes}}\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_composed_function}{Composed\+Function}}\+: Represents composition of two functions.
\item {\ttfamily \doxylink{class_add_function}{Add\+Function}}\+: Represents the addition of two functions.
\item {\ttfamily \doxylink{class_multiply_function}{Multiply\+Function}}\+: Represents the multiplication of two functions.
\item {\ttfamily \doxylink{class_exponential_function}{Exponential\+Function}}\+: Represents exponentiation of a function (supports both integer and floating-\/point powers).
\item {\ttfamily \doxylink{class_identity_function}{Identity\+Function}}\+: Represents the identity function (returns the input as output).
\item {\ttfamily \doxylink{class_constant_function}{Constant\+Function}}\+: Represents a constant function (always returns the same value).
\item {\ttfamily \doxylink{class_function_wrapper}{Function\+Wrapper}}\+: Allows wrapping of standard C++ functions (e.\+g., lambdas, function pointers) as {\ttfamily \doxylink{class_abstract_function}{Abstract\+Function}}.
\end{DoxyItemize}
\end{DoxyItemize}
\item {\bfseries{\doxylink{class_parser}{Parser} Class}}
\begin{DoxyItemize}
\item {\bfseries{Tokenization}}\+: Converts the input string into tokens representing numbers, operators, and functions.
\item {\bfseries{Parsing}}\+: Builds the AST by respecting operator precedence (e.\+g., addition, multiplication, parentheses).
\item {\bfseries{Function Generation}}\+: Generates callable functions from the AST nodes (e.\+g., {\ttfamily sin}, {\ttfamily cos}, operators like {\ttfamily +}, {\ttfamily \texorpdfstring{$\ast$}{*}}).
\item {\bfseries{Final Function}}\+: Allows retrieval of the final generated function after parsing and function generation.
\end{DoxyItemize}
\end{DoxyEnumerate}
